Previous Invoice Number

Moving accounting to Odoo and want to map invoice numbers from old invoicing system?
This module will allow you to:
- save invoice numbers from other system and view them next to each other
- display old invoice number in the printed Invoice(pdf) if exists so your customer will be able to map them
- see the old invoice number on the tree view(list view) next to Odoo Invoice numbers

Previous Invoice No available for Invoices and Bills

A new field is added for both Customer Invoices as well as to Vendor Bills List Views(Tree Views).

Invoices/Bills Form Views

New field is placed for easy access and comparison with the Odoo Invoice number.
At some point it is likely that you will set the numbers to be the same in both accounting systems.
This position will make your work easier
Form view is the place where you enter the old Invoice numbers.

PDF / Prints updated

The old Invoice number will appear on the Invoice print(pdf) - only if it exists.
It looks better and makes INvoice less crowded.

Installation and Configuration

Module is safe and easy to use. After installation you need to go to Settings / Reports / Reports and search for "Invoice".
Uncheck the box "Reload from Attachment"
