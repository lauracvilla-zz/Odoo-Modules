Calendar Invitations for everyone

Now you can access the additional "Invitations" tab that will allow you to:
- access the customer(attendee) details with couple clicks
- set the attendee status(Accepted, Uncertain, Decline)
- send mail
The Invitations tab is visible for all users now. Module code contains detailed description so you can understand how it works if you want to. Originally it was hidden and visible only in developers mode.

Send mail & change meeting status

Change the status of the meeting with one click. You can also simply send an email.

Go to partner form

You can open the partner form with one click. No need to go to the "contacts" and search for the partner.
