Signup redirect to shop

This is a simple module that redirects freshly signed user to the shop. It simply changes the view of /web/signup site so after filling the signup form user will be redirected to the shop. You can change page it is redirecting to by editing the xml file in views, it can be easily done using any text editor.

How it works

Redirect is added to the sign up button (t-att-value="redirect or '/shop'"). Change the "shop" to any other site on your website if you like.
