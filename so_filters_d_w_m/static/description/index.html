Quotations and Orders Filters Today, Last Week, Last Month

This module adds three new filters to the Orders list as well as to Quotations list.

Thanks to this user will be able to filter Sales Orders basing on the creation date (Today, within Last Week or Within last 30 days)

How it works

New filters will be added to the filter list, on the list view in Quotations as well as in Orders.
