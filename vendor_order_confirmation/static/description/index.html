RFQ - Vendor Confirmation Box

Many companies requirement is to have the Request for Quotation signed physically by Vendor.
This module adds the additional table, where Vendor can confirm the Quotation.

Additional table details

New table has some a standardised description so it can be used with most of the companies.
Company name, phone and email are dynamic and comes from the actual company (if your company details will be changed in Odoo this fields will automatically update).
Module is easy to update and adjust for your business needs.
