Salesperson phone number on Sales Order and Quotation Print

Make life easier for your Customers as well as for the Staff with this simple module. Module will display the phone number to the Salesperson responsible for the sale so customer can contact him directly. Simple module, serious time saver!

Remember to set the phone number of your Salespersons

You can easily do it by editing the Salesperson data in Contacts. Want to add mobile number or email? Not a problem - module is simple to edit and all necessary comments are available inside.
