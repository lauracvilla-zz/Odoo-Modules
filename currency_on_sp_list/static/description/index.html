Currency on Supplier Pricelist

When dealing with suppliers (vendors) from different countries knowing the currency might be more than useful. This module adds currency to the Supplier Pricelist.

How it works

Currency will be added next to the price on the Pricelist. Remember to activate "Allow using and importing vendor pricelists" in order to see the pricelist menu.
