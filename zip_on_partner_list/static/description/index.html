Zip on Contacts List / Tree View

This simple module adds Zip code to the contacts list (tree view). It is really powerful feature as it is possible now to sort contacts by the post code.

See the Zip code in the Contacts tab

Simply open the contacts tab and choose the "list view" to see the module in action.

See the Zip code in the Sales/Partners

Now, when you navigate to the Sales / Partners you will be able to see Partner's Zip on the tree view.

See the Zip code in the Purchases / Vendors

Now, when you navigate to the Purchases / Vendors you will be able to see Vendor Zip on the tree view.

Invoices / Accounting Section

Zip code is now visible everywhere, where customer/partner contacts are listed.
