Website shop hide price

This module hides the price of the products from not logged in users. It user is not logged in he will also see the "Log in to see price" button instead of "Add to Cart". See the screenshots below to see the module in action.

Hide Price on the Kanban View

If user is not logged in he won't see the prices in the classic view.

Once user is logged in the prices will be visible.

Hide Price on the List View

If user is not logged in he won't see the prices in the list view.

Once user is logged in the prices will be visible.

Hide Price in the Product Description page

If user is not logged in he won't see the price in the product description.
Instead of "Add to Basket" user will see the "Log in to see price" button. This might convince user to log in or register.

Once user is logged in the prices will be visible.

Hints & More

In order to "Log in to see price" button work correctly remember to give users opportunity to sign up. You can do it by activating developer mode and entering the "Settings".

You can also change the button description and link using the "Edit HTML" on the website.

Module was tested on the clean Odoo version. If you are using any custom theme the module might not work correctly (please contact for more details)
